### 23. SemiAnalysis 洞察 Token 经济：200 美元 AI 订阅榨出 70 倍用量
- SemiAnalysis 购买 Anthropic 和 OpenAI 全部订阅方案，模拟高强度编码任务直至触及每周上限。
- 月费 200 美元的 Claude Max 20x 方案，按 API 价格换算最高可消耗约值 8000 美元的 token。
- ChatGPT Pro 20x 方案对应最高约值 14000 美元的 token。
- 用户通过订阅可获取 40-70 倍的 API 价值。
- 该机构指出这种价格体系在重度用户持续榨满上限后可能难以长期维持。

> 为什么值得关注：这是首次有第三方机构用"端到端实测"把订阅制 vs API 制的套利空间量化。70 倍意味着 Claude / OpenAI 的订阅定价在算力成本侧严重低于用户实际消耗——如果这种"补贴"持续 6-12 个月，大厂有动机调价；如果被资本市场问到，订阅制本身的可持续性将成为下一波财报焦点。

来源：IT 之家 · SemiAnalysis Token 经济报告
