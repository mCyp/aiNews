## 💡 技巧

**1. 驾驭 Claude Code：CLAUDE.md / 规则 / 技能 / 子代理 / 钩子 / 输出样式 / 附加系统提示七种自定义方式如何选**
CLAUDE.md 永远加载（构建命令、编码规范），路径范围规则避免无关上下文消耗，技能按需调用共享 token 预算，子代理隔离上下文并行执行，钩子绕过压缩做确定性自动化（如 linter/聊天记录备份）。按"加载时机 × 压缩行为 × 上下文成本"四象限选择，避免一项用到底浪费 token。
- [Claude Blog](https://claude.com/blog/steering-claude-code-skills-hooks-rules-subagents-and-more)

**2. cuTile Rust：内存安全、无数据竞争的 GPU 内核系统**
NVIDIA 开源基于 tile 的 GPU 编程系统，可用 Rust 写无数据竞争的内核。`#[cutile::module]` 宏将内核 AST 嵌入主机二进制，运行时经 CUDA Tile IR JIT 编译为 GPU cubin。在 B200 上逐元素操作 7 TB/s（约 91% 峰值带宽）、GEMM 2 PFlop/s（约 92% 密集 f16 峰值）。基于此构建的 Grout 推理引擎在 RTX 5090 解码 Qwen3-4B 达 171 tokens/s。
- [GitHub](https://github.com/nvlabs/cutile-rs)

**3. OpenClaw / SillyTavern 接入 OpenRouter：一条命令统一 300+ 模型**
OpenClaw 内置 OpenRouter 支持，一条命令即可配置统一密钥、统一账单并实现跨 300+ 模型自动故障转移。SillyTavern 同样支持：单个 OpenRouter API Key 即可解锁 300+ 模型（含免费模型）做角色扮演。两份教程均带常见错误修复。
- [OpenRouter 博客 - OpenClaw](https://openrouter.ai/blog/tutorials/openclaw-openrouter)
