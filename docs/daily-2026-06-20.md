## 💡 技巧

**1. OpenRouter vs Portkey：团队 LLM 网关选型指南**
OpenRouter 是托管路由网络（70+ 供应商/300+ 模型、5.5% 平台费、前 100 万免费、零数据保留）；Portkey 是 AI 控制平面（HIPAA/SSO/私有部署、$49/月起）。两者可串联使用。
- [OpenRouter Blog](https://openrouter.ai/blog/insights/openrouter-vs-portkey)

**2. OpenRouter vs LiteLLM：自托管还是托管网关？**
OpenRouter 托管 Cloudflare 边缘（5.5% 平台费），LiteLLM 自部署（Docker/PG/Redis，免费但需基础设施）。模型月支出超 $3,600 起自托管更划算；LiteLLM 提供 6 种路由策略 + 自定义 Python 路由。
- [OpenRouter Blog](https://openrouter.ai/blog/insights/openrouter-vs-litellm)

**3. baoyu-design Skill 迭代循环：用户反馈 → Agent 修复 → 测试覆盖**
宝玉分享迭代经验：用户测试发现导出样式/渐变丢失问题，让 Agent 复现+分析+出方案+补测试，修复后效果改善。Skill 可导出含 AI 配图的 PPTX，在 PowerPoint/Keynote 二次编辑。
- [宝玉推文](https://x.com/dotey/status/2068042001895809420)

**4. Elasticsearch 持久化 Agent 内存层：R@10 = 0.89**
记忆分情景/语义/程序三类，各存独立索引、设不同写速率与过期。召回采用 BM25 + Jina v5 稠密向量 RRF 融合，再经交叉编码器重排。168 道 QA 评估 R@10 平均 0.89，零跨租户泄漏；MCP 协议客户端可访问。
- [Elastic Blog](https://www.elastic.co/search-labs/blog/agent-memory-elasticsearch)

**5. Salesforce CodeGen 端到端代码生成工作流**
