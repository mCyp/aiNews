Anthropic 把智能体循环分四类：turn-based（用户提示触发）、goal-based（`/goal` 设定可验证完成标准与最大轮次）、time-based（`/loop` 按时间间隔或 `/schedule` 移至云端）、proactive（基于事件或计划自动运行）；附 SKILL.md 把人工验证步骤编码进 Claude 自检流程，减少 turn-based 循环中的人工操作。
- [Claude Blog](https://claude.com/blog/getting-started-with-loops)

**2. Every 复利工程：单人维护 5 款产品，80% 时间不写代码**
媒体公司 Every 开源"复利工程"方法论：四步循环 Plan→Work→Review→Compound，每次解决问题的解法写入 CLAUDE.md 和 docs/solutions/，使 AI 下次自动避坑；工程师 80% 时间花在 Plan 和 Review，20% 写代码；配套开源插件支持 Claude Code 等，含 26 个专项 agent、23 条工作流命令、13 项技能；`/workflows:review` 一次并发 14 个 agent 审查代码，`/workflows:plan` 在 ultrathink 模式下并发 40+ 研究 agent。
- [小互](https://x.com/xiaohu/status/2071796715162857477)

**3. Vibe Coding 必备的两个 Prompt：第一性原理 + 对抗式审查**
"从第一性原理出发"强制 AI 跳出类比推理，从基本事实重新推导本质；"对抗式审查"让 AI 站在恶意用户角度攻防式审查代码，检出 OOM 死循环、未来时间污染等手工难发现的 BUG；两个 Prompt 形成生成与验证闭环，纯 Vibe Coding 项目 AIHOT 最近一周请求量超千万次。
- [卡兹克](https://mp.weixin.qq.com/s/umPqTD_-IubbhXIgiS47eQ)

**4. Anthropic 工程师 Margot Van Laar：提示词工程实战，评估是唯一严谨方式**
Anthropic 应用 AI 工程师在 Code with Claude 分享：大部分时间在调试和维护已有生产提示词；客服机器人维护用 XML 标签结构化清理，移除旧模型遗留的"禁止列表"指令（新模型会过度拟合），精确计算应调用工具，转人工决策需明确代价与收益；零售排班 Agent 从零构建时，拆成生成-评估-修复三个简单提示词更稳定，选用更强推理模型（Opus）。她反复强调——评估（Eval）是唯一严谨方式，没有评估就是碰运气。
- [Berry Xia](https://x.com/berryxia/status/2071610700213191075)

**5. Claude Code 6 个实用 Hook 玩法：让 AI 从聊天框变事件驱动系统**
Claude Code 内置近 30 个 Hook 事件（年初仅 13 个），运行时不消耗 token。6 个实用玩法：权限弹窗提醒、开机日程播报（问候+天气+飞书日程）、上下文预压缩时自动生成摘要卡片、结合 Skill 自动整理下载文件夹、启动后每小时久坐提醒、通过 Bark 实现手机/手表任务完成/失败推送。
- [卡兹克](https://mp.weixin.qq.com/s/LVj2foSXi_hBRKxjuYaUyw)

**6. 小互开源个人 IP 配图技能"小互 IP Studio"：31 个原创角色**
包含 31 个原创角色（15 个手绘线稿角色 + 16 个谐音梗 meme 形象）及一套配图方法论。Agent 自动读取文章、规划配图类型（情绪图/示意图/四格漫画）、生成并自查返工；默认手绘线稿淡彩，备 5 种皮肤（3D 盲盒、黑白线稿等）；安装仅需 Python3，支持 Claude Code、Codex 等工具。
- [小互](https://x.com/xiaohu/status/2070317717811540149)

**7. 一条命令在 HuggingFace Jobs 启动 vLLM 服务器**
`hf jobs run` 命令 + `vllm/vllm-openai` 镜像 + GPU flavor + 暴露端口 8000 + 超时即可启动；通过 OpenAI 兼容 API 访问，每次请求需携带 HF token 作为 bearer token（仅限有读权限的用户）；示例部署 Qwen/Qwen3-4B，多 GPU 需 `--tensor-parallel-size`；`a10g-large` $1.50/小时按分钟计费，`hf jobs cancel` 停止。
- [Hugging Face](https://huggingface.co/blog/vllm-jobs)

**8. NVIDIA NeMo AutoModel：一行 import 加速 MoE 模型微调**
基于 Transformers v5 的开源库，添加 Expert Parallelism、DeepEP 融合 all-to-all 调度、TransformerEngine 内核。MoE 微调训练吞吐量提升 3.4-3.7 倍，GPU 内存减少 29-32%，仅需改动一行 import；16 节点 128 张 H100 全微调 Nemotron 3 Ultra 550B A55B，v5 因内存不足无法运行，AutoModel 凭借 EP=64 专家并行使训练可行。
- [Hugging Face](https://huggingface.co/blog/nvidia/accelerating-fine-tuning-nvidia-nemo-automodel)

**9. Weave 模型路由工具：本地代理自动选最佳模型**
通过 `npx @workweave/router` 安装，作为本地代理运行在 localhost:8080；采用基于 Avengers-Pro 1 的集群评分器，每个请求自动选择最佳模型；支持 Anthropic/OpenAI/Gemini 原生 API，并通过 OpenRouter 接入 DeepSeek/Kimi/GLM/Qwen/Llama/Mistral 等开源模型；兼容 Claude Code/Codex/Cursor 客户端；提供 OTLP 追踪，支持自托管部署。
- [GitHub](https://github.com/workweave/router)

**10. Leaf 实时通话 AI 分身：工程延迟压到 1 秒内**
开源项目将网红峰哥做成实时通话 AI 分身：语音识别用 Cartesia ink-whisper 降噪防误触发；大模型选 MiniMax 高速版，首字响应 361ms；语音合成用 VoxCPM 开源克隆，15 秒素材即可复刻；整体从 8-20 秒优化到体感 2-3 秒；人格通过女娲 Skill 从直播语料蒸馏口头禅和思维逻辑；普通人半小时可跑通。
- [阿易 AI Notes](https://x.com/AYi_AInotes/status/2070531964067623381)
